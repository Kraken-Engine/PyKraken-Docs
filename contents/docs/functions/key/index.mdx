## Is Just Pressed
<div className="api-card">
<ApiSig name="is_just_pressed" params={[{ name: "scancode", type: "Scancode" }]} returns="bool" />

_**Args**_

- `scancode` : The physical key.

_**Returns**_

`bool` : True if the key was newly pressed.
