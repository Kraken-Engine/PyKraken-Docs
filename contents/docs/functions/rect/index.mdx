## Clamp
<div className="api-card">
<ApiSig name="clamp" params={[{ name: "rect", type: "Rect" }, { name: "min", type: "Vec2" }, { name: "max", type: "Vec2" }]} returns="Rect" />

Clamp a rectangle to be within the specified bounds.

_**Args**_

- `rect` : The rectangle to clamp.
- `min` : The minimum bounds as (min_x, min_y).
- `max` : The maximum bounds as (max_x, max_y).

_**Returns**_

`Rect` : A new rectangle clamped within the bounds.

_**Raises**_

- `ValueError` : If min &gt;= max or rectangle is larger than the clamp area.

---

<ApiSig name="clamp" params={[{ name: "rect", type: "Rect" }, { name: "other", type: "Rect" }]} returns="Rect" />

Clamp a rectangle to be within another rectangle.

_**Args**_

- `rect` : The rectangle to clamp.
- `other` : The rectangle to clamp within.

_**Returns**_

`Rect` : A new rectangle clamped within the other rectangle.

_**Raises**_

- `ValueError` : If rect is larger than the clamp area.

</div>

## Scale By
<div className="api-card">
<ApiSig name="scale_by" params={[{ name: "rect", type: "Rect" }, { name: "factor", type: "SupportsFloat" }]} returns="Rect" />

Scale a rectangle by a uniform factor.

_**Args**_

- `rect` : The rectangle to scale.
- `factor` : The scaling factor (must be &gt; 0).

_**Returns**_

`Rect` : A new rectangle scaled by the factor.

_**Raises**_

- `ValueError` : If factor is &lt;= 0.

---

<ApiSig name="scale_by" params={[{ name: "rect", type: "Rect" }, { name: "factor", type: "Vec2" }]} returns="Rect" />

Scale a rectangle by different factors for width and height.

_**Args**_

- `rect` : The rectangle to scale.
- `factor` : The scaling factors as (scale_x, scale_y).

_**Returns**_

`Rect` : A new rectangle scaled by the factors.

_**Raises**_

- `ValueError` : If any factor is &lt;= 0.

</div>

## Scale To
<div className="api-card">
<ApiSig name="scale_to" params={[{ name: "rect", type: "Rect" }, { name: "size", type: "Vec2" }]} returns="Rect" />

Scale a rectangle to the specified size.

_**Args**_

- `rect` : The rectangle to scale.
- `size` : The new size as (width, height).

</div>
