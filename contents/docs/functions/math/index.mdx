## Remap
<div className="api-card">
<ApiSig name="remap" multiline params={[{ name: "in_min", type: "SupportsFloat" }, { name: "in_max", type: "SupportsFloat" }, { name: "out_min", type: "SupportsFloat" }, { name: "out_max", type: "SupportsFloat" }, { name: "value", type: "SupportsFloat" }]} returns="float" />

Remap a value from one range to another.

_**Args**_

- `in_min` : Input range minimum.
- `in_max` : Input range maximum.
- `out_min` : Output range minimum.
- `out_max` : Output range maximum.
- `value` : The value to remap.

_**Returns**_

`float` : The remapped value in the output range.

_**Raises**_

- `ValueError` : If in_min equals in_max.
