## Clamp
<div className="api-card">

<ApiSig name="clamp" params={[{ name: "vec", type: "Vec2" }, { name: "min", type: "Vec2" }, { name: "max", type: "Vec2" }]} returns="Vec2" />

_**Parameters**_

`float` : The clamped value.

## Lerp
<div className="api-card">

<ApiSig name="lerp" params={[{ name: "a", type: "Vec2" }, { name: "b", type: "Vec2" }, { name: "t", type: "float" }]} returns="Vec2" />

Linearly interpolate between two vectors.

_**Parameters**_

- `a` : The starting vector.
- `b` : The ending vector.

_**Parameters**_

_**Returns**_

`float` : The interpolated float.

</div>

## Remap

Remap a value from one range to another.
