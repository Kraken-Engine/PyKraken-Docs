## From Polar
<div className="api-card">
<ApiSig name="from_polar" params={[{ name: "angle", type: "float" }, { name: "radius", type: "float" }]} returns="Vec2" />

## Clamp
<div className="api-card">
<ApiSig name="clamp" multiline params={[{ name: "vec", type: "Vec2" }, { name: "min_vec", type: "Vec2" }, { name: "max_vec", type: "Vec2" }]} returns="Vec2" />

Clamp a vector between two boundary vectors.

_**Args**_

- `vec` : The vector to clamp.
- `min_vec` : The minimum boundary vector.
- `max_vec` : The maximum boundary vector.

_**Returns**_

`Vec2` : A new vector with components clamped between min and max.

- `max_val` : The maximum boundary.

_**Returns**_

`float` : The clamped value.

</div>

## Remap
<div className="api-card">
<ApiSig name="remap" multiline params={[{ name: "in_min", type: "float" }, { name: "in_max", type: "float" }, { name: "out_min", type: "float" }, { name: "out_max", type: "float" }, { name: "value", type: "float" }]} returns="float" />

Remap a value from one range to another.

_**Args**_

- `in_min` : Input range minimum.
- `in_max` : Input range maximum.
- `out_min` : Output range minimum.
- `out_max` : Output range maximum.
- `value` : The value to remap.

_**Returns**_

`float` : The remapped value in the output range.

_**Raises**_

- `ValueError` : If in_min equals in_max.

</div>
