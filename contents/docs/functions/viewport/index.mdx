## Layout
<div className="api-card">
<ApiSig name="layout" multiline params={[{ name: "count", type: "SupportsInt" }, { name: "mode", type: "ViewportMode", default: "ViewportMode.VERTICAL" }]} returns="list[Rect]" />

Layout the screen into multiple viewports.
The viewports are created with the current renderer target resolution in mind.

_**Args**_

- `count` : The number of viewports to create (between 2 and 4).
- `mode` : The layout mode for 2 viewports (VERTICAL or HORIZONTAL).
                              Defaults to VERTICAL.

_**Returns**_

`list[Rect]` : A list of Rects representing the viewports.

</div>

## Set
<div className="api-card">
<ApiSig name="set" params={[{ name: "rect", type: "Rect" }]} returns="None" />

Set the current viewport to the given rectangle.

_**Args**_

- `rect` : The rectangle defining the viewport.

</div>

## Unset
<div className="api-card">
<ApiSig name="unset" returns="None" />

Unset the current viewport, reverting to the full rendering area.

</div>
