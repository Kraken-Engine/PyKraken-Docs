## Close
<div className="api-card">
<ApiSig name="close" returns="None" />

Close the window.

Marks the window as closed, typically used to signal the main loop to exit.
This doesn't destroy the window immediately but sets the close flag.

</div>

## Create
<div className="api-card">
<ApiSig name="create" multiline params={[{ name: "title", type: "str" }, { name: "width", type: "SupportsInt" }, { name: "height", type: "SupportsInt" }]} returns="None" />

Create a window with the requested title and resolution.

_**Args**_

- `title` : Non-empty title no longer than 255 characters.
- `width` : The window width, must be positive.
- `height` : The window height, must be positive.

_**Raises**_

- `RuntimeError` : If a window already exists or SDL window creation fails.
- `ValueError` : If the title is invalid or any dimension is non-positive.

</div>

## Set Title
<div className="api-card">
<ApiSig name="set_title" params={[{ name: "title", type: "str" }]} returns="None" />

Set the title of the window.
