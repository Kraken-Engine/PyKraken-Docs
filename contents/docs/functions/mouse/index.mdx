## Get Pos
<div className="api-card">

<ApiSig name="get_pos" returns="Vec2" />

Get the current position of the mouse cursor.

_**Returns**_

`Vec2` : The current mouse position as a 2D vector.

</div>

## Get Rel
<div className="api-card">

<ApiSig name="get_rel" returns="Vec2" />

Get the relative mouse movement since the last frame.

_**Returns**_

`Vec2` : The relative movement of the mouse as a 2D vector.

</div>

## Is Pressed
<div className="api-card">

<ApiSig name="is_pressed" params={[{ name: "button", type: "MouseButton" }]} returns="bool" />

Check if a mouse button is currently pressed.

_**Parameters**_

- `button` : The mouse button to check.

_**Returns**_

`bool` : True if the button is currently pressed.

</div>

## Is Just Pressed
<div className="api-card">

<ApiSig name="is_just_pressed" params={[{ name: "button", type: "MouseButton" }]} returns="bool" />

Check if a mouse button was pressed this frame.

_**Parameters**_

- `button` : The mouse button to check.

_**Returns**_

`bool` : True if the button was just pressed.

</div>

## Is Just Released
<div className="api-card">

<ApiSig name="is_just_released" params={[{ name: "button", type: "MouseButton" }]} returns="bool" />

Check if a mouse button was released this frame.

_**Parameters**_

- `button` : The mouse button to check.

_**Returns**_

`bool` : True if the button is currently pressed.

</div>

## Lock
<div className="api-card">

<ApiSig name="lock" returns="None" />

Lock the mouse to the center of the window.

Useful for first-person controls where you want to capture mouse movement
without letting the cursor leave the window area.

</div>

## Unlock
<div className="api-card">

<ApiSig name="unlock" returns="None" />

Unlock the mouse from the window, allowing it to move freely.

</div>

## Is Locked
<div className="api-card">

<ApiSig name="is_locked" returns="bool" />

Check if the mouse is currently locked to the window.

_**Returns**_

`bool` : True if the mouse is locked.

</div>

## Hide
<div className="api-card">

<ApiSig name="hide" returns="None" />

Hide the mouse cursor from view.

The cursor will be invisible but mouse input will still be tracked.

</div>

## Show
<div className="api-card">

<ApiSig name="show" returns="None" />

Show the mouse cursor if it was hidden.

</div>

## Is Hidden
<div className="api-card">

<ApiSig name="is_hidden" returns="bool" />

Check if the mouse cursor is currently hidden.

_**Returns**_

`bool` : True if the cursor is hidden.

</div>
