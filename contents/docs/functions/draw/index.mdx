## Circle
<div className="api-card">
<ApiSig name="circle" multiline params={[{ name: "circle", type: "Circle" }, { name: "color", type: "Color" }, { name: "thickness", type: "SupportsFloat", default: "0" }, { name: "num_segments", type: "SupportsInt", default: "36" }]} returns="None" />

Draw a circle to the renderer.

_**Args**_

- `circle` : The circle to draw.
- `color` : The color of the circle.
- `thickness` : The line thickness. If &lt;= 0 or &gt;= radius, draws filled circle. Defaults to 0 (filled).
- `num_segments` : Number of segments to approximate the circle.
                                  Higher values yield smoother circles. Defaults to 36.

</div>

</div>

## Geometry
<div className="api-card">
<ApiSig name="geometry" multiline params={[{ name: "texture", type: "Texture | None" }, { name: "vertices", type: "Sequence[Vertex]" }, { name: "indices", type: "Any", default: "None" }]} returns="None" />

Draw arbitrary geometry using vertices and optional indices.

_**Args**_

- `texture` : The texture to apply to the geometry. Can be None.
- `vertices` : A list of Vertex objects.
- `indices` : A list of indices defining the primitives.
                                   If None or empty, vertices are drawn sequentially.

</div>

## Points From Ndarray
<div className="api-card">
<ApiSig name="points_from_ndarray" multiline params={[{ name: "points", type: "Annotated[numpy.ArrayLike, numpy.float64]" }, { name: "color", type: "Color" }]} returns="None" />

Batch draw points from a NumPy array.

This fast path accepts a contiguous NumPy array of shape (N,2) (dtype float64) and
reads coordinates directly with minimal overhead. Use this to measure the best-case
zero-copy/buffer-backed path.

_**Args**_

- `points` : Array with shape (N,2) containing x,y coordinates.
- `color` : The color of the points.

_**Raises**_

- `ValueError` : If the array shape is not (N,2).
- `RuntimeError` : If point rendering fails.

</div>
