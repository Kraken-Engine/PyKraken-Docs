## Clear
<div className="api-card">

<ApiSig name="clear" params={[{ name: "color", type: "Color", default: "None" }]} returns="None" />

Clear the current render target with the specified color.

_**Parameters**_

- `color` : The color to clear with. If omitted, black is used.

---

<ApiSig name="clear" params={[{ name: "r", type: "int" }, { name: "g", type: "int" }, { name: "b", type: "int" }, { name: "a", type: "int", default: "255" }]} returns="None" />

Clear the current render target with the specified RGBA color components.

_**Parameters**_

- `r` : Red component (0-255).
- `g` : Green component (0-255).
- `b` : Blue component (0-255).
- `a` : Alpha component (0-255), defaults to 255 (opaque).

</div>

## Present
<div className="api-card">

<ApiSig name="present" returns="None" />

## Get Res
<div className="api-card">

<ApiSig name="get_res" returns="Vec2" />

Get the current resolution of the renderer's output target.

_**Returns**_

</div>
