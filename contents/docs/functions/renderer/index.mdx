## Clear
<div className="api-card">

<ApiSig name="clear" params={[{ name: "color", type: "Color", default: "None" }]} returns="None" />

Clear the current render target with the specified color.

_**Parameters**_

- `color` : The color to clear with. If omitted, black is used.

---

<ApiSig name="clear" params={[{ name: "r", type: "int" }, { name: "g", type: "int" }, { name: "b", type: "int" }, { name: "a", type: "int", default: "255" }]} returns="None" />

Clear the current render target with the specified RGBA color components.

_**Parameters**_

- `r` : Red component (0-255).
- `g` : Green component (0-255).
- `b` : Blue component (0-255).
- `a` : Alpha component (0-255), defaults to 255 (opaque).

</div>

## Present
<div className="api-card">

<ApiSig name="present" returns="None" />

Present the current render target to the display, swapping buffers.

</div>

## Draw
<div className="api-card">

<ApiSig name="draw" params={[{name: "texture", type: "Texture"}, {name: 'dst', type: 'Rect'}, {name: 'src', type: 'Rect', default: 'None'}]} returns="None" />

Render the texture within the specified destination rectangle, optionally using a source rectangle to select a portion of the texture.

_**Parameters**_
- `texture` : The texture to draw.
- `dst` : The destination rectangle to render the texture into.
- `src` : An optional source rectangle to select a portion of the texture.

---

<ApiSig name="draw" params={[{name: "texture", type: "Texture"}, {name: 'pos', type: 'Vec2', default: 'None'}, {name: 'anchor', type: 'Anchor', default: 'CENTER'}]} returns="None" />

Render the texture at the specified position, optionally using an anchor point to control the alignment.

_**Parameters**_
- `texture` : The texture to draw.
- `pos` : The position to render the texture.
- `anchor` : The anchor point for alignment.

</div>

## Get Res
<div className="api-card">

<ApiSig name="get_res" returns="Vec2" />

Get the current resolution of the renderer's output target.

_**Returns**_

`Vec2` : The width and height of the render target in pixels.

</div>

## Read Pixels
<div className="api-card">

<ApiSig name="read_pixels" params={[{ name: "src", type: "Rect", default: "None" }]} returns="PixelArray" />

Read pixel data from the current render target within the specified rectangle.

_**Parameters**_

- `src` : The source rectangle to read pixels from. If omitted, the entire render target is read.

_**Returns**_

`PixelArray` : The pixel data read from the render target.

</div>
