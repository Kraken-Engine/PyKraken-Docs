## Flip
<div className="api-card">

`flip(pixel_array: PixelArray, flip_x: bool, flip_y: bool) → PixelArray`

Flip a pixel array horizontally and/or vertically.

_**Parameters**_

- `pixel_array` : The pixel array to flip.
- `flip_x` : Whether to flip the pixel array horizontally.
- `flip_y` : Whether to flip the pixel array vertically.

_**Returns**_

`PixelArray` : The flipped pixel array.

</div>
