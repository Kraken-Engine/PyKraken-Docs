## Rotate By
<div className="api-card">
<ApiSig name="rotate_by" multiline params={[{ name: "delta", type: "SupportsFloat" }, { name: "clockwise", type: "bool", default: "True" }, { name: "dur", type: "SupportsFloat", default: "0.0" }, { name: "ease", type: "Any", default: "None" }]} returns="Effect" />

Create a rotate-by effect.

_**Args**_

- `delta` : Delta angle in radians to rotate by in radians.
- `clockwise` : Direction of rotation. True for clockwise, False for counterclockwise.
- `dur` : Duration in seconds.
- `ease` : Easing function (t -&gt; t).

_**Returns**_

`Effect` : The rotate-by effect.

</div>
