## Is Pressed
<div className="api-card">

<ApiSig name="is_pressed" params={[{ name: "button", type: "GamepadButton" }, { name: "slot", type: "int", default: "0" }]} returns="bool" />

Check if a gamepad button is currently being held down.

_**Parameters**_

- `button` : The gamepad button to check.
- `slot` : The gamepad slot to check (default is 0).

_**Returns**_

`bool` : True if the button is pressed, otherwise False.

</div>

## Set Deadzone
<div className="api-card">
