## Get Connected Slots
<div className="api-card">
<ApiSig name="get_connected_slots" returns="list[int]" />

Get a list of connected gamepad slot indices.

_**Returns**_

`list[int]` : A list of slot IDs with active gamepads.

</div>

## Get Deadzone
<div className="api-card">
<ApiSig name="get_deadzone" params={[{ name: "slot", type: "SupportsInt", default: "0" }]} returns="float" />

Get the current dead zone value for a gamepad's analog sticks.

_**Args**_

- `slot` : Gamepad slot ID (default is 0).

_**Returns**_

`float` : Deadzone threshold.

</div>

## Get Left Stick
<div className="api-card">
<ApiSig name="get_left_stick" params={[{ name: "slot", type: "SupportsInt", default: "0" }]} returns="Vec2" />

Get the left analog stick position.

_**Args**_

- `slot` : Gamepad slot ID (default is 0).

_**Returns**_

`Vec2` : A vector of stick input normalized to [-1, 1], or (0, 0) if inside dead zone.

</div>

## Is Just Pressed

## Set Deadzone
<div className="api-card">
<ApiSig name="set_deadzone" multiline params={[{ name: "deadzone", type: "SupportsFloat" }, { name: "slot", type: "SupportsInt", default: "0" }]} returns="None" />

Set the dead zone threshold for a gamepad's analog sticks.

_**Args**_

- `deadzone` : Value from 0.0 to 1.0 where movement is ignored.
- `slot` : Gamepad slot ID (default is 0).

</div>
