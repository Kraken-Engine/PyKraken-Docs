## Poll
<div className="api-card">

<ApiSig name="poll" returns="list[Event]" />

Poll for all pending user input events.

This clears input states and returns a list of events that occurred since the last call.

_**Returns**_

`list[Event]` : A list of events that have occurred since the last call to `poll()`.

</div>

## Schedule
<div className="api-card">

<ApiSig name="schedule" params={[{ name: "event", type: "Event" }, { name: "delay_ms", type: "int" }, { name: "repeat", type: "bool" }]} returns="None" />

Schedule an event to be triggered after a delay.
Calling this function on the same event again will replace its past schedule.

_**Parameters**_

- `event` : The custom event to schedule.
- `delay_ms` : The delay in milliseconds before the event is triggered.
- `repeat` : Whether the event should be repeated.

</div>

<div className="api-card">
