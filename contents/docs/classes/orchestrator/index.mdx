## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `finalized` | Whether the orchestrator has been finalized. | <code>bool</code> |
| `playing` | Whether the animation is currently playing. | <code>bool</code> |
| `finished` | Whether the animation has completed. | <code>bool</code> |
| `looping` | Whether the animation should loop when finished. | <code>bool</code> |

### Finalize
<div className="api-card">
<ApiSig name="finalize" returns="None" />

Finalize the orchestrator, preventing further edits.

Must be called before play(). Logs a warning if called multiple times.

</div>

### Play
<div className="api-card">
<ApiSig name="play" returns="None" />

Start playing the animation from the beginning.

Logs a warning if not finalized or if there are no steps.

</div>

### Pause
<div className="api-card">
<ApiSig name="pause" returns="None" />

Pause the animation at the current position.

</div>

### Resume
<div className="api-card">
<ApiSig name="resume" returns="None" />

Resume a paused animation.

</div>

### Stop
<div className="api-card">
<ApiSig name="stop" returns="None" />

Stop the animation and reset to the beginning.

</div>

### Rewind
<div className="api-card">
<ApiSig name="rewind" returns="None" />

Reset the animation to the beginning without stopping.

</div>
