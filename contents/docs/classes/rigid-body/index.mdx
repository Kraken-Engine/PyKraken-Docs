## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `angular_damping` | The angular damping of the body. | <code>float</code> |
| `angular_velocity` | The angular velocity of the body. | <code>float</code> |
| `fixed_rotation` | Whether the body has fixed rotation. | <code>bool</code> |
| `is_awake` | Whether the body is currently awake and simulating. | <code>bool</code> |
| `is_bullet` | Whether CCD is enabled for this body. | <code>bool</code> |
| `linear_damping` | The linear damping of the body. | <code>float</code> |
| `linear_velocity` | The linear velocity of the body. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `mass` | The mass of the body. | <code>float</code> |

### Apply Angular Impulse
<div className="api-card">
<ApiSig name="apply_angular_impulse" params={[{ name: "impulse", type: "SupportsFloat" }, { name: "wake", type: "bool", default: "True" }]} returns="None" />

Apply an angular impulse to the body.

_**Args**_

- `impulse` : The angular impulse to apply.
- `wake` : Whether to wake the body if it's sleeping. Defaults to True.

</div>

### Apply Force
<div className="api-card">
<ApiSig name="apply_force" multiline params={[{ name: "force", type: "Vec2" }, { name: "point", type: "Vec2" }, { name: "wake", type: "bool", default: "True" }]} returns="None" />

Apply a force to the body at a specific point.

_**Args**_

- `force` : The force vector to apply.
- `point` : The point (in world coordinates) where the force is applied.
