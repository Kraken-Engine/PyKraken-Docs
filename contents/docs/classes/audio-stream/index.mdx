## Methods
---

### Pause
<div className="api-card">
<ApiSig name="pause" returns="None" />

Pause the audio stream playback.

The stream position is preserved and can be resumed with resume().

</div>

### Play
<div className="api-card">
<ApiSig name="play" multiline params={[{ name: "fade_in_ms", type: "SupportsInt", default: "0" }, { name: "loop", type: "bool", default: "False" }, { name: "start_time_seconds", type: "SupportsFloat", default: "0.0" }]} returns="None" />

Play the audio stream with optional fade-in time, loop setting, and start position.

Starts playback from the specified time position. If the stream is already
playing, it will restart from the specified position.

_**Args**_

- `fade_in_ms` : Fade-in duration in milliseconds. Defaults to 0.
- `loop` : Whether to loop the audio continuously. Defaults to False.
- `start_time_seconds` : Time position in seconds to start playback from. Defaults to 0.0.

</div>

### Resume
<div className="api-card">
<ApiSig name="resume" returns="None" />

Resume paused audio stream playback.

Continues playback from the current stream position.

</div>

### Rewind
<div className="api-card">
<ApiSig name="rewind" returns="None" />

Rewind the audio stream to the beginning.

Sets the playback position back to the start of the audio file. Does not affect
the current play state (playing/paused).

</div>

### Seek
<div className="api-card">
<ApiSig name="seek" params={[{ name: "time_seconds", type: "SupportsFloat" }]} returns="None" />

Seek to a specific time position in the audio stream.

Sets the playback position to the specified time in seconds. Does not affect
the current play state (playing/paused).

_**Args**_

- `time_seconds` : The time position in seconds to seek to.

</div>

### Set Looping
<div className="api-card">
<ApiSig name="set_looping" params={[{ name: "loop", type: "bool" }]} returns="None" />

Set whether the audio stream loops continuously.

_**Args**_

- `loop` : True to enable looping, False to disable.

</div>

### Stop
<div className="api-card">
<ApiSig name="stop" params={[{ name: "fade_out_ms", type: "SupportsInt", default: "0" }]} returns="None" />

Stop the audio stream playback.

_**Args**_

- `fade_out_ms` : Fade-out duration in milliseconds. If 0, stops immediately.
                              If &gt; 0, fades out over the specified duration. Defaults to 0.

</div>
