## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `text` | The string content of the text object. | `str` |
| `color` | The color of the text. | `Color` |
| `wrap_width` | The width at which to wrap text. Set to 0 for no wrapping. | `int` |
| `size` | The size of the text in pixels. | `Vec2` |
| `width` | The width of the text in pixels. | `int` |

### Draw
<div className="api-card">

<ApiSig name="draw" params={[{name: "pos", type: "Vec2", default: "None"}, {name: "anchor", type: "Anchor", default: "Anchor.TOP_LEFT"}]} returns="None" />

Draw text to the renderer.
`\n` will write text on a new line.

_**Parameters**_

- `pos` : The position to draw the text at. Defaults to (0, 0).
- `anchor` : The anchor point to draw the text from. Defaults to TOP_LEFT.

</div>
