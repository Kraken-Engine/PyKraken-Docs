---
title: TextProperties
description: TextProperties holds styling for text objects on the map. Access via the 'tilemap' submodule.
---


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `font_family` | Font family name. | <code>str</code> |
| `pixel_size` | Font size in pixels. | <code>int</code> |
| `wrap` | Whether text wrapping is enabled. | <code>bool</code> |
| `color` | Text color. | <code><a href="/docs/classes/color">Color</a></code> |
| `bold` | Bold style flag. | <code>bool</code> |
| `italic` | Italic style flag. | <code>bool</code> |
| `underline` | Underline style flag. | <code>bool</code> |
| `strikethrough` | Strikethrough style flag. | <code>bool</code> |
| `kerning` | Kerning enabled flag. | <code>bool</code> |
| `align` | Horizontal text alignment. | <code><a href="/docs/manual/constants#align">Align</a></code> |
| `text` | Text content. | <code>str</code> |
