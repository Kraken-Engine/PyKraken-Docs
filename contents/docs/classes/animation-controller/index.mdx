---
title: AnimationController
description: A sprite animation manager and controller.
---

## Constructor
<div className="api-card">
<ApiSig name="AnimationController" />
Load animations with `load_sprite_sheet` before using.
</div>

## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `current_animation_name` | The current animation's name | `str` |
| `clip` | The source rectangle (clip) for the current animation frame | `Rect` |
| `playback_speed` | The playback speed multiplier | `float` |
| `frame_index` | The current frame index of the animation | `int` |
| `progress` | The progress through the current animation as a value between 0.0 and 1.0 | `float` |
| `looping` | Whether the animation should loop after playing | `bool` |

## Methods
---

### Load Sprite Sheet
<div className="api-card">

<ApiSig name="load_sprite_sheet" multiline params={[{ name: "frame_size", type: "Vec2" }, { name: "strips", type: "list[SheetStrip]" }]} returns="None" />

Load animation frames for a sprite sheet.
Strips are read horizontally from left to right, top to bottom, as provided in the strips list.

_**Parameters**_
- `frame_size` : The size of each frame in the sprite sheet.
- `strips` : A list of sheet strips to load from the sprite sheet.

</div>

### Set
<div className="api-card">

<ApiSig name="set" params={[{ name: "name", type: "str" }]} returns="None" />

Set the current animation by name without affecting playback state.

_**Parameters**_
- `name` : The name of the animation to set as current.

</div>

### Play
<div className="api-card">

<ApiSig name="play" params={[{ name: "name", type: "str" }]} returns="None" />

Play an animation from the beginning by name.

_**Parameters**_
- `name` : The name of the animation to play.

</div>

### Play From
<div className="api-card">

<ApiSig name="play_from" params={[{ name: "frame_index", type: "int" }]} returns="None" />

Play an animation from a specific frame.

_**Parameters**_
- `frame_index` : The index of the frame to start playing from.

</div>

### Is Finished
<div className="api-card">

<ApiSig name="is_finished" returns="bool" />

Check if the current animation has finished a loop since the last frame update.

_**Returns**_

`bool` : `True` if the animation is finished, otherwise `False`.

</div>

### Rewind
<div className="api-card">

<ApiSig name="rewind" returns="None" />

Rewind the current animation to the first frame.

</div>

### Pause
<div className="api-card">

<ApiSig name="pause" returns="None" />

Pause the current animation.

</div>

### Resume
<div className="api-card">

<ApiSig name="resume" returns="None" />

Resume the current animation.

</div>
