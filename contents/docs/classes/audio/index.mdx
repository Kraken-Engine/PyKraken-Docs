---
title: Audio
description: Abstract base class for all audio resources. Access via the 'mixer' submodule.
---


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `playing` | True if currently playing. | <code>bool</code> |
| `volume` | Volume scalar (0.0 to 1.0). | <code>float</code> |

## Methods
---

### Play
<div className="api-card">
<ApiSig name="play" params={[{ name: "fade_in", type: "SupportsFloat", default: "0.0" }]} returns="None" />

Start audio playback.

_**Args**_

- `fade_in` : Fade in duration in seconds. Defaults to 0.0.

</div>

### Stop
<div className="api-card">
<ApiSig name="stop" params={[{ name: "fade_out", type: "SupportsFloat", default: "0.0" }]} returns="None" />

Stop all playing instances of this audio.

_**Args**_

- `fade_out` : Fade out duration in seconds. Defaults to 0.0.

</div>
