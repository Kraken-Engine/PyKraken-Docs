---
title: PrismaticJoint
description: A joint that constrains two bodies to move only along a specified axis. Access via the 'physics' submodule.
---

<Note type="warning" title="Experimental API">
  The physics submodule is a VERY experimental and new API that is highly susceptible to breaking changes in the future.
</Note>

Inherits from [Joint](/docs/classes/joint).


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `spring_enabled` | Whether the spring is enabled. | <code>bool</code> |
| `spring_hz` | The spring frequency in Hertz. | <code>float</code> |
| `spring_damping_ratio` | The spring damping ratio. | <code>float</code> |
| `target_translation` | The target translation for the motor. | <code>float</code> |
| `limit_enabled` | Whether the translation limits are enabled. | <code>bool</code> |
| `motor_enabled` | Whether the motor is enabled. | <code>bool</code> |
| `motor_speed` | The target motor speed. | <code>float</code> |
| `max_motor_force` | The maximum motor force. | <code>float</code> |
| `lower_limit` | The lower translation limit. | <code>float</code> |
| `upper_limit` | The upper translation limit. | <code>float</code> |
| `motor_force` | The current motor force. | <code>float</code> |
| `translation` | The current joint translation. | <code>float</code> |
| `speed` | The current joint translation speed. | <code>float</code> |
