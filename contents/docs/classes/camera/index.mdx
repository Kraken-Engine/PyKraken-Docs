## Constructor
<div className="api-card">

- <ApiSig name="Camera" params={[{ name: "set_active", type: "bool", default: "False" }]} returns="Camera" />

Represents a 2D camera used for rendering.
</div>

## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `angle` | Get or set the camera angle in radians. | <code>float</code> |
| `local_pos` | Get or set the camera position in camera-local space. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `world_pos` | Get or set the camera's world position. | <code><a href="/docs/classes/vec2">Vec2</a></code> |

## Methods
---

### Move Local
<div className="api-card">
<ApiSig name="move_local" params={[{ name: "delta", type: "Vec2" }]} returns="None" />

Move the camera by a delta in camera-local (screen) space.

_**Args**_

- `delta` : Local movement delta.

</div>

### Move World
<div className="api-card">
<ApiSig name="move_world" params={[{ name: "delta", type: "Vec2" }]} returns="None" />

Move the camera by a delta in world space.

_**Args**_

- `delta` : World movement delta.

</div>

### Screen To World
<div className="api-card">
<ApiSig name="screen_to_world" params={[{ name: "screen_pos", type: "Vec2" }]} returns="Vec2" />

Convert a screen position to a world position using this camera's translation.

_**Args**_

- `screen_pos` : The screen position to convert.

_**Returns**_

`Vec2` : The resulting world position.

</div>

### Set
<div className="api-card">
<ApiSig name="set" returns="None" />

Set this camera as the active one for rendering.

Only one camera can be active at a time.

</div>

### Unset
<div className="api-card">
<ApiSig name="unset" returns="None" />

Unset this camera as the active one for rendering.

</div>

### World To Screen
<div className="api-card">
<ApiSig name="world_to_screen" params={[{ name: "world_pos", type: "Vec2" }]} returns="Vec2" />

Convert a world position to a screen position using this camera's translation.

_**Args**_

- `world_pos` : The world position to convert.

_**Returns**_

`Vec2` : The resulting screen position.

</div>
