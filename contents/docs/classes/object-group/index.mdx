---
title: ObjectGroup
description: ObjectGroup is a layer containing placed MapObjects. Access via the 'tilemap' submodule.
---

Inherits from [Layer](/docs/classes/layer).


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `color` | Tint color for non-tile objects. | <code><a href="/docs/classes/color">Color</a></code> |
| `opacity` | Layer opacity from 0.0 to 1.0. | <code>float</code> |
| `draw_order` | Drawing order for objects in the group. | <code><a href="/docs/manual/constants#object-group-draw-order">ObjectGroup.DrawOrder</a></code> |
| `objects` | MapObjectList of objects in the group. | <code>MapObjectList</code> |

## Methods
---

### Draw
<div className="api-card">
<ApiSig name="draw" multiline params={[{ name: "angle", type: "float", default: "0.0" }, { name: "pivot", type: "Vec2", default: "..." }]} returns="None" />

Draw the object group.

_**Args**_

- `angle` : Rotation angle in degrees. Defaults to 0.0.
- `pivot` : Rotation pivot as normalized coordinates relative to the map size. Defaults to (0.5, 0.5).
