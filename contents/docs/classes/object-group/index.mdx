## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `color` | Tint color for non-tile objects. | <code><a href="/docs/classes/color">Color</a></code> |
| `draw_order` | Drawing order for objects in the group. | <code><a href="/docs/classes/object-group">ObjectGroup</a>.DrawOrder</code> |
| `objects` | List of objects in the group. | <code>MapObjectList</code> |
| `opacity` | Layer opacity from 0.0 to 1.0. | <code>float</code> |
