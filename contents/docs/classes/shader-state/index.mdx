## Constructor
<div className="api-card">
<ApiSig name="ShaderState" multiline params={[
    { name: "fragment_file_path", type: "str" },
    { name: "uniform_buffer_count", type: "int", default: "0" }
]} />

_**Parameters**_
- `fragment_file_path` : Path to the fragment shader file.
- `uniform_buffer_count` : Number of uniform buffers used by the shader. Default is 0.
</div>

<div className="api-card">

### Set Uniform
<div className="api-card">

<ApiSig name="set_uniform" params={[{name: 'binding', type: 'int'}, {name: 'data', type: 'Buffer | bytes'}]} returns="None" />

Set the uniform variable in the shader.

_**Parameters**_

- `binding` : The binding point of the uniform variable.
- `data` : The data to set the uniform variable to.
