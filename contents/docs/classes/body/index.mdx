## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `collision_layer` | The body's collision layer (category bits). | <code>int</code> |
| `collision_mask` | The body's collision mask. | <code>int</code> |
| `is_valid` | Indicates whether the body is not destroyed. | <code>bool</code> |
| `pos` | The position of the body in world coordinates. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `rotation` | The rotation of the body in radians. | <code>float</code> |

## Methods

### Add Collider
<div className="api-card">
<ApiSig name="add_collider" multiline params={[{ name: "circle", type: "Circle" }, { name: "density", type: "SupportsFloat", default: "1.0" }, { name: "friction", type: "SupportsFloat", default: "0.2" }, { name: "restitution", type: "SupportsFloat", default: "0.0" }, { name: "enable_events", type: "bool", default: "False" }, { name: "is_sensor", type: "bool", default: "False" }]} returns="None" />

Add a circular collider to the body.

_**Args**_

- `circle` : The circular shape to add as a collider.
- `density` : The density of the collider. Defaults to 1.0.
- `friction` : The friction coefficient of the collider. Defaults to 0.2.
- `restitution` : The restitution (bounciness) of the collider. Defaults to 0.0.
- `enable_events` : Whether to enable hit events for this collider. Defaults to False.
- `is_sensor` : Whether the collider is a sensor. Defaults to False.
