---
title: Body
description: Base class for all physics bodies. Access via the 'physics' submodule.
---

<Note type="warning" title="Experimental API">
  The physics submodule is an experimental and new API that is highly susceptible to breaking changes in the future.
</Note>


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `pos` | The position of the body in world coordinates. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `rotation` | The rotation of the body in radians. | <code>float</code> |
| `collision_layer` | The body's collision layer (category bits). | <code>int</code> |
| `collision_mask` | The body's collision mask. | <code>int</code> |
| `is_valid` | Indicates whether the body is not destroyed. | <code>bool</code> |
