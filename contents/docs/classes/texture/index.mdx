## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `alpha` | Get or set the alpha modulation of the texture as a float between `0.0` and `1.0`. | <code>float</code> |
| `clip_area` | Get or set the clip area (atlas region) of the texture. | <code><a href="/docs/classes/rect">Rect</a></code> |
| `flip` | The flip settings for horizontal and vertical mirroring. | <code><a href="/docs/classes/texture-flip">Texture.Flip</a></code> |
| `height` | The height of the texture in pixels. | <code>int</code> |
| `size` | The dimensions of the texture as a `Vec2`. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `tint` | Get or set the color tint applied to the texture. | <code><a href="/docs/classes/color">Color</a></code> |
| `usage` | The usage flags describing how the texture can be used. | <code><a href="/docs/manual/constants#texture-usage">TextureUsage</a></code> |
| `width` | The width of the texture in pixels. | <code>int</code> |

### Get Rect
<div className="api-card">
<ApiSig name="get_rect" returns="Rect" />

Return a Rect with position (0, 0) and the texture's dimensions.

_**Returns**_

`Rect` : A rectangle representing the texture's bounds.

</div>

### Has Usage
<div className="api-card">
<ApiSig name="has_usage" params={[{ name: "usage", type: "TextureUsage" }]} returns="bool" />

Check whether the texture was created with a specific usage flag.

_**Args**_

- `usage` : Usage flag to test.

_**Returns**_

`bool` : True if the usage flag is present.

</div>

### Make Additive
<div className="api-card">
<ApiSig name="make_additive" returns="None" />

Set the texture to use additive blending mode.

In additive mode, the texture's colors are added to the destination,
creating bright, glowing effects.

</div>

### Make Multiply
<div className="api-card">
<ApiSig name="make_multiply" returns="None" />

Set the texture to use multiply blending mode.

In multiply mode, the texture's colors are multiplied with the destination,
creating darkening and shadow effects.

</div>

### Make Normal
<div className="api-card">
<ApiSig name="make_normal" returns="None" />

Set the texture to use normal (alpha) blending mode.

This is the default blending mode for standard transparency effects.

</div>
