    ]} />

_**Parameters**_
- `file_path` : Path to the image file to load.
- `pixel_array` : The pixel array to convert to a texture.
- `size` : The width and height of the empty texture to create.
- `scale_mode` : The scaling mode to use for the texture.
- `access` : The access mode for the texture.
</div>

## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `flip` | Whether the texture is flipped horizontally and/or vertically | `Flip` |
| `size` | The original width and height of the texture | `Vec2` |
| `alpha` | The alpha modulation of the texture (0.0 to 1.0) | `float` |
| `tint` | The color tint applied to the texture during rendering | `Color` |
