## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `points` | The list of Vec2 points that define the polygon vertices. | <code>list[<a href="/docs/classes/vec2">Vec2</a>]</code> |
| `centroid` | Get or set the centroid of the polygon. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `perimeter` | Get the perimeter of the polygon. | <code>float</code> |
| `area` | Get the area of the polygon. | <code>float</code> |
| `is_convex` | Check if the polygon is convex. | <code>bool</code> |
| `is_concave` | Check if the polygon is concave. | <code>bool</code> |
