## Methods

### Clamp
<div className="api-card">
<ApiSig name="clamp" params={[{ name: "other", type: "Rect" }]} returns="None" />

Clamp this rectangle to be within another rectangle.

_**Args**_

- `other` : The rectangle to clamp within.

_**Raises**_

- `ValueError` : If this rectangle is larger than the clamp area.

---

<ApiSig name="clamp" params={[{ name: "min", type: "Vec2" }, { name: "max", type: "Vec2" }]} returns="None" />

Clamp this rectangle to be within the specified bounds.

_**Args**_

- `min` : The minimum bounds as (min_x, min_y).
- `max` : The maximum bounds as (max_x, max_y).

_**Raises**_

- `ValueError` : If min &gt;= max or rectangle is larger than the clamp area.

</div>

### Inflate
<div className="api-card">
<ApiSig name="inflate" params={[{ name: "offset", type: "Vec2" }]} returns="None" />

Inflate the rectangle by the given offset.

The rectangle grows in all directions. The position is adjusted to keep the center
in the same place.

_**Args**_

- `offset` : The amount to inflate by as (dw, dh).

</div>
