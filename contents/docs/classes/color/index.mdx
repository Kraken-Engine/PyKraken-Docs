_**Parameters**_
- `hex` : Hex color string (with or without '#' prefix).
- `rgba` : A sequence of 3 or 4 integers [0-255] (RGB or RGBA).
- `r` : Red value [0-255].
- `g` : Green value [0-255].
- `b` : Blue value [0-255].
- `a` : Alpha value [0-255]. Defaults to 255.
