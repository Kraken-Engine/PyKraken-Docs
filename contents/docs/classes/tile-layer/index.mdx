---
title: TileLayer
description: TileLayer represents a grid of tiles within the map. Access via the 'tilemap' submodule.
---


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `opacity` | Layer opacity from 0.0 to 1.0. | <code>float</code> |
| `tiles` | List of tiles in the layer grid. | <code><a href="/docs/classes/tile-layer">TileLayer</a>.TileLayerTileList</code> |

## Methods
---

### Get From Area
<div className="api-card">
<ApiSig name="get_from_area" params={[{ name: "area", type: "Rect" }]} returns="TileLayer.TileResultList" />

Return tiles intersecting a Rect area.

_**Args**_

- `area` : World-space area to query.

_**Returns**_

`list[TileLayer.TileResult]` : List of TileResult entries for tiles intersecting the area.

</div>

### Get From Point
<div className="api-card">
<ApiSig name="get_from_point" params={[{ name: "position", type: "Vec2" }]} returns="Any" />

Return the tile at a given world position.

_**Args**_

- `position` : World-space position to query.

_**Returns**_

`Optional[TileLayer.TileResult]` : TileResult entry if a tile exists at the position, None otherwise.

</div>
