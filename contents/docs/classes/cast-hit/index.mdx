---
title: CastHit
description: Result of a ray cast or shape cast query. Access via the 'physics' submodule.
---

<Note type="warning" title="Experimental API">
  The physics submodule is a VERY experimental and new API that is highly susceptible to breaking changes in the future.
</Note>


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `body` | The body that was hit. | <code><a href="/docs/classes/body">Body</a></code> |
| `point` | The point of the hit in world coordinates. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `normal` | The normal vector of the hit surface. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `fraction` | The fraction along the cast path at which the hit occurred. | <code>float</code> |
