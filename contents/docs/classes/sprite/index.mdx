## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `clip` | Source rectangle for texture sampling (None = full texture). | <code>Any</code> |
| `texture` | The sprite's texture. | <code><a href="/docs/classes/texture">Texture</a></code> |
| `transform` | The sprite's transform. | <code><a href="/docs/classes/transform">Transform</a></code> |
| `velocity` | The sprite's velocity. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `visible` | Whether the sprite is visible. | <code>bool</code> |
