## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `pos` | The center position of the circle as a Vec2. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `radius` | The radius of the circle. | <code>float</code> |
| `diameter` | Get or set the diameter of the circle. | <code>float</code> |
| `left` | Get or set the x coordinate of the leftmost point of the circle. | <code>float</code> |
| `right` | Get or set the x coordinate of the rightmost point of the circle. | <code>float</code> |
| `top` | Get or set the y coordinate of the topmost point of the circle. | <code>float</code> |
| `bottom` | Get or set the y coordinate of the bottommost point of the circle. | <code>float</code> |
| `area` | Get the area of the circle. | <code>float</code> |
| `circumference` | Get the circumference of the circle. | <code>float</code> |
