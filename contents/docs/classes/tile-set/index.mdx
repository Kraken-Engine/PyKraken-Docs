---
title: TileSet
description: TileSet represents a collection of tiles and associated metadata. Access via the 'tilemap' submodule.
---


## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `first_gid` | First global tile id (GID) in this tileset. | <code>int</code> |
| `last_gid` | Last global tile id (GID) in this tileset. | <code>int</code> |
| `name` | Tileset name. | <code>str</code> |
| `tile_size` | Size of tiles in pixels. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `spacing` | Pixel spacing between tiles in the source image. | <code>int</code> |
| `margin` | Pixel margin around the source image. | <code>int</code> |
| `tile_count` | Total number of tiles in the tileset. | <code>int</code> |
| `columns` | Number of tile columns in the source image. | <code>int</code> |
| `tile_offset` | Per-tile offset applied when rendering. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `terrains` | TerrainList of terrain definitions. | <code>TileSet.TerrainList</code> |
| `tiles` | TileSetTileList of tile metadata entries. | <code>TileSet.TileSetTileList</code> |
| `texture` | Source texture for the tileset. | <code><a href="/docs/classes/texture">Texture</a></code> |

### Has Tile
<div className="api-card">
<ApiSig name="has_tile" params={[{ name: "id", type: "int" }]} returns="bool" />
