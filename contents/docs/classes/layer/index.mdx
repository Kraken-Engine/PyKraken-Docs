## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `visible` | Whether the layer is visible. | <code>bool</code> |
| `offset` | Per-layer drawing offset. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `opacity` | Layer opacity from 0.0 to 1.0. | <code>float</code> |
| `name` | Layer name. | <code>str</code> |
| `type` | Layer type enum. | <code><a href="/docs/manual/constants#layer-type">LayerType</a></code> |

## Methods
---

### Draw
<div className="api-card">
<ApiSig name="draw" params={[{ name: "angle", type: "float", default: "0.0" }, { name: "pivot", type: "Vec2", default: "..." }]} returns="None" />

Draw the layer to the current renderer.

_**Args**_

- `angle` : Rotation angle in degrees. Defaults to 0.0.
- `pivot` : Rotation pivot as normalized coordinates relative to the map size. Defaults to (0.5, 0.5).
