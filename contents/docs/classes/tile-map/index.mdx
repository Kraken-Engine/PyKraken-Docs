title: TileMap
description: A Tiled map containing multiple layers of tiles.

## Constructor
<div className="api-card">
<ApiSig name="TileMap" params={[{ name: "tmx_path", type: "str" }, { name: "border_size", type: "int", default: "0" }]} />

_**Parameters**_
- `tmx_path` : Path to the TMX file.
- `border_size` : Optional border (in pixels) around each tile in the tileset. Defaults to 0.
</div>

## Methods
---

### Get Layer
<div className="api-card">

<ApiSig name="get_layer" params={[{name: 'name', type: 'str'}, {name: 'type', type: 'TileLayer.Type', default: 'TileLayer.TILE'}]} returns="TileLayer" />

Get a layer object from the tile map by name and type.

_**Returns**_

`TileLayer` : The layer object with the specified name and type.

</div>

### Get Layers
<div className="api-card">

<ApiSig name="get_layers" returns="list[TileLayer]" />

Get all layers in the tile map.

_**Returns**_

`list[TileLayer]` : A list of all layers in the tile map.

</div>

### Get Tile Collection
<div className="api-card">

<ApiSig name="get_tile_collection" params={[{name: 'layers', type: 'Sequence[TileLayer]'}]} returns="list[Tile]" />

Get all tiles from the specified layers.

_**Returns**_

`list[Tile]` : A list of all tiles from the specified layers.

</div>
