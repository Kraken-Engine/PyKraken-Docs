## Properties
<hr style={{marginBottom: 0}} />

| Name | Description | Type |
| --- | --- | --- |
| `background_color` | The background color of the element (if no texture is provided). | <code><a href="/docs/classes/color">Color</a> &#124; None</code> |
| `texture` | The texture to use for the element. | <code><a href="/docs/classes/texture">Texture</a></code> |
| `slice` | The region of the texture to use. | <code><a href="/docs/classes/rect">Rect</a></code> |
| `offset` | The offset of the element within its parent. | <code><a href="/docs/classes/vec2">Vec2</a></code> |
| `font` | The font to use for text rendering. | <code><a href="/docs/classes/font">Font</a></code> |
| `text_color` | The color of the text. | <code><a href="/docs/classes/color">Color</a> &#124; None</code> |
| `padding` | The padding around the element. | <code>float</code> |
| `margin` | The margin around the element. | <code>float</code> |
| `gap` | The gap between child elements. | <code>float</code> |
| `border_width` | The width of the border. | <code>int</code> |
| `border_radius` | The radius of the border corners. | <code>float</code> |
| `border_color` | The color of the border. | <code><a href="/docs/classes/color">Color</a> &#124; None</code> |
| `width` | The width of the element. | <code>float &#124; None</code> |
| `height` | The height of the element. | <code>float &#124; None</code> |
